计算机科学是什么？研究信息的学科：怎么表示信息、处理信息。
本质是解决问题的思维，编程只是工具。

信息的表示。用手指最多只能数到 5；电脑内部晶体管只有开/关两种状态，
开=1，关=0，所以电脑用二进制（binary）。1 位（bit）就是一个 0 或 1；
1 字节（byte）= 8 位，可以表示 1 个英文字母。

人类规定了映射关系：ASCII 里每个字母（A、B、C……）对应一个数字，
数字背后又是一串 0 和 1。

计算机只懂 0 和 1，但人类不可能手搓 0 和 1 去和它交流，于是有了编程语言。

C 语言距离 0/1 很近，抽象程度低，要自己处理底层细节，对用户不友好；
但换来控制力和速度，操作系统就是 C 写的。

Python 距离硬件最远，抽象程度最高，细节被藏得最多，所以好懂、友好；
缺点是控制力和速度不如 C。Python 的解释器恰好是用 C 写的。

人写的语言电脑听不懂，这时候就需要**编译器**：把源代码翻译成 0 和 1 
的可执行文件。

语言的进步就是封装细节供人复用。写 Python 不用懂 C，写 C 不用懂 0/1。
我们应该 stand on the shoulders of others——站在别人的肩膀上，
就能造出复杂的东西。
